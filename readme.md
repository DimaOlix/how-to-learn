# Проектная работа №2: Научиться учиться.
***
**Данный проект - вэб-страница. Она написана мной в рамках прохождения первого курса Яндекс.Практикум по программе вэб-разработчик.**

***Эта страница рассказывает посетителю о некоторых существующих техниках, методах и принципах обучения. Приводит реальные статистические факты относящиеся к данной теме. Также затрагивает и возможные проблемы методик обучения применяемых в большинстве учебных заведений.***

***На странице приведены интересные видео ролики направленные на улучшение процесса обучения. Рекомендуется книга и полезные интернет-ресурсы.
При написании данной страницы использовались:***
* ***flex-box***
* ***position***
* ***iframe***
* ***анимация с использованием transform в сочетании с transition***
* ***анимация с использованием keyframes.***

***В проекте выполнена классов блоков и элементов по методологии БЭМ. Организация файловой структуры по Nested БЭМ.***
